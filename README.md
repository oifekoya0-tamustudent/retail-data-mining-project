# Retail Data Mining Project

## Overview
This project analyzes customer purchase behavior using transactional retail data.
The objective is to uncover purchasing patterns and segment customers using data mining techniques.

## Techniques Used

### Course Techniques
- Frequent Itemset Mining (Apriori)
- Association Rule Mining
- KMeans Clustering

### Beyond-Course Techniques
- Sequential pattern mining (proposed)
- Advanced customer lifetime value modeling (future work)

## Dataset
Online Retail Dataset (UCI Machine Learning Repository)
https://archive.ics.uci.edu/ml/datasets/Online+Retail

Subset of data used for computational efficiency.

## Project Structure
- Data Cleaning and Preprocessing
- Exploratory Data Analysis
- Market Basket Analysis
- Customer Segmentation
- Initial Research Direction

## Key Insights
- Sales are dominated by a small number of products.
- Customer purchasing behavior forms distinct clusters.
- Strong association rules reveal product bundling opportunities.
